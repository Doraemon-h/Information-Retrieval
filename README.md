# Information-Retrieval
使用python实现简单的信息检索系统，实现了基于tf-idf的TopK查询，布尔查询，短语查询三种查询方式，同时也实现了词干提取，堆排序，拼写检查等功能等功能，在实现布尔查询中，类比数据结构中运算法则的时候的处理，将表达式转化为后缀表达式处理。
